### 计算二维数组行列数

在 Python 中，如果遇到二维数组的问题了，需要格外注意：计算二维数组的行列的时候绝对不能出错：

```py
# 3 行 2 列
>>> alist = [[1, 2],[3, 4], [5, 6]]
>>> len(alist)
$ 3
>>> len(alist[0])
$ 2
```

所以一般的计算方式为：

```py
# alist
rows = len(alist)
cols = len(alist[0])
```
